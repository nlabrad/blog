# nicolab.dev

Personal blog focused on software development, technical troubleshooting, and system administration.

## About

This blog covers topics including:
- Linux system administration
- Development workflows and tools
- Hardware compatibility and fixes
- Technical deep-dives

## Built With

- [Jekyll](https://jekyllrb.com/) - Static site generator
- GitHub Pages - Hosting
- Custom retro terminal theme

## Local Development

```bash
bundle install
bundle exec jekyll serve --livereload
```

Visit `http://localhost:4000` to preview locally.
